Welcome to my personal portfolio! This project showcases my skills, projects, and experience as a Frontend Web Developer. Built with HTML, CSS, and JavaScript, it features a modern design with micro-interactions, a downloadable resume, and links to my social profiles.

## Overview
This portfolio highlights my journey as a developer, including my education, projects, skills, and certifications. It’s designed to be responsive, interactive, and visually appealing, with hover effects and animations to enhance user experience.

## Features
- **Responsive Design:** Adapts seamlessly to mobile, tablet, and desktop screens.
- **Navigation:** Sticky nav bar with smooth scrolling and active link highlighting.
- **Home Section:** Profile picture, bio, and buttons for "View My Work" and "View Resume."
- **About Section:** Accordion-style details on education, experience, and interests.
- **Projects Section:** Interactive flip cards showcasing projects with GitHub links.
- **Certificates Section:** Display of my META Front-End Certification.
- **Skills Section:** Animated progress bars for frontend, backend, and tools.
- **Contact Section:** Functional contact form (requires EmailJS setup) and contact info.
- **Footer:** Social media icons with brand-colored hover effects.
- **Micro-Interactions:** Hover effects, click animations, and scroll-triggered animations.

## Technologies
- **HTML5:** Structure and semantics.
- **CSS3:** Styling, animations, and responsive layouts.
- **JavaScript:** Interactivity, smooth scrolling, and form handling.
- **Font Awesome:** Social media icons.
- **EmailJS:** Contact form integration (requires configuration).
- **Google Fonts:** Poppins font for typography.

## Usage
- **Navigation: Click nav links to scroll to sections.**
- **Projects: Hover over project cards to flip and reveal details.**
- **Resume: Click "View Resume" to open the PDF in a new tab.**
- **Contact: Fill out the form and submit (requires EmailJS setup).**
- **Social Links: Click footer icons to visit my profiles.**
## Customization
- **Profile Info: Edit #bio, #about, and #contact sections in index.html.**
- **Projects: Add or modify .project-card entries in the #projects section.**
- **Skills: Adjust .skill entries and data-width values in #skills.**
- **Colors: Change CSS variables or specific colors in styles.css (e.g., #00ccff for primary blue).**
- **Images: Replace assets in assets/ with your own (keep filenames consistent or update src attributes).**
